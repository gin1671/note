困ったところ

  • railsコマンドですべてエラーになる。 rails aborted! NoMethodError: undefined method `file' for #<Pry::History:0x00007fd1b79c9538> Did you mean? filter `block in make_coolline': undefined method `file' for #<Pry::History:0x00007fe6db27de30> (NoMethodError) gemのcoollineをコメントアウトしたら通った。不要そうなので削除した。

  • db:createやrails sの失敗。 bundle exec rails db:create Access denied for user 'root'@'localhost' (using password: YES) Couldn't create 'tweet-reader_dev' database. Please check your configuration. rails aborted! Mysql2::Error::ConnectionError: Access denied for user 'root'@'localhost' (using password: YES)

database.ymlではパスワード設定をしているのに、実際は設定されていなかった。 mysql -u root でMySQLモードになるので（-pなし）。パスワードがないとわかった。

MySQL8以降では設定方法が変わったらしい。 mysql> USE mysql; mysql> ALTER USER 'root'@'localhost' identified BY '*****';

学び

  • 1つ1つ丁寧に、理解を得ながら進める。
